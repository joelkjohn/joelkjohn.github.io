Care

This project was done for my final year project which was very important to me. within this website, there are components of PHP, HTML/CSS, JavaScript, Python, SQL, and some elements of Bootstrap.

the idea of creating care was to provide a platform for those who are in need in terms of certain mental health issues such as anxiety and stress. the platform would generally provide users with features such as journalling which is known to help someone overcome stress and anxiety, a chatbot which was made to have similar characteristics as a befriender, and finally a video and quote feature to provide users ways and methods to overcome some issues and also provide the users some inspiration to keep them going.

Within this project i would say i have improved in the design as compared to the hospital system which was my first project. moreover i learned some things such as integrating a python script by the implementation of PHP Curl which made the python chatbot react similarly as an API. In terms of design i learned how to make the interface more professional as compared to my first project and it is more realistic as well. however, there are many things that i can improve on in terms of design when it comes to making it appear more professional and pleasing to the eye.

Hospital Appointment Booking System

This was done as an assignment for a subject called Web Technology which was also my first project and experience within web development and design. It mainly consists components of HTML, CSS, PHP, Bootstrap, and finally a little JavaScript. As for the database it mainly runs on SQL.

The system generally allows patients to book appointments according to availability of locations and treatments. As for the doctors side, they would be allowed to manage their appointments.

From this project, I have managed to learn how to use certain components such as tables, forms, buttons, placement of images, and how to display data. I also managed to learn how to integrate a specific location for google maps using the embedded link which is provided on the internet.
